🎓 Career Point University Chatbot

AI-Powered Career Guidance and Student Support System

📌 Project Overview

The Career Point University Chatbot is an AI-driven web application designed to assist students with academic and career-related queries. It provides instant responses, guidance on courses, admissions, career paths, and university information through an interactive conversational interface.

This chatbot aims to simplify access to information and improve the user experience by offering real-time support, personalized recommendations, and automated assistance.

🚀 Features
💬 AI Chatbot Interface
Provides real-time answers to student queries.
🎓 Course & Admission Guidance
Helps students explore programs, eligibility, and admission processes.
🧭 Career Recommendations
Suggests career paths based on interests and academic background.
📚 University Information Support
Provides details about departments, facilities, and services.
⚡ Instant Response System
Reduces waiting time compared to manual support.
🌐 User-Friendly Interface
Simple and responsive design for students and visitors.
